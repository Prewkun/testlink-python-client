# PFS MES Procedure Parameters Reference

This document provides a compact Req/Opt parameter view for TestLink procedures.
Use it as a quick reference while refactoring request builders and validation logic.

## Routing and Transaction Procedures

PfsVerifyUserInput
Req: REQUEST_TYPE, DATABASE, USER_ID, PASSWORD
Opt: PRODUCTION_ORDER, OPERATION_CODE, WI_OPERATION, ITEM_NUMBER, WORK_CENTER

PfsQuery
Req: REQUEST_TYPE, DATABASE, USER_ID, PASSWORD, OPERATION_CODE, SERIAL_NUMBER
Opt: PRODUCTION_ORDER, ITEM_NUMBER, RETURN_VALUES, OVERRIDE_OK, MULTIPLE_PO

PfsSendResults
Req: REQUEST_TYPE, DATABASE, USER_ID, PASSWORD, OPERATION_CODE, SERIAL_NUMBER, PASS_FAIL
Opt: PRODUCTION_ORDER, ITEM_NUMBER, WORK_CENTER, HISTORY_COMMENT, OVERRIDE_OK, MULTIPLE_PO, FAIL_REQUIRES_DEFECT, DEFECT_FIELDS, DEFECTS, custom test-data parameters

PfsSendSignoff
Req: REQUEST_TYPE, DATABASE, USER_ID, PASSWORD, OPERATION_CODE, SERIAL_NUMBER
Opt: PRODUCTION_ORDER, ITEM_NUMBER, WORK_CENTER, HISTORY_COMMENT, OVERRIDE_OK, MULTIPLE_PO

PfsPanelize
Req: REQUEST_TYPE, DATABASE, USER_ID, PASSWORD, (PRODUCTION_ORDER or ITEM_NUMBER), OPERATION_CODE, SERIAL_NUMBER
Opt: PANEL_NUMBER, PANEL_NUMBER_COMMENT, WORK_CENTER, HISTORY_COMMENT, RETURN_VALUES

PfsLinkCompData
Req: REQUEST_TYPE, DATABASE, USER_ID, PASSWORD, (PRODUCTION_ORDER or ITEM_NUMBER), OPERATION_CODE, SERIAL_NUMBER, REF_DES, COMP_SERIAL_NUMBER
Opt: COMP_ITEM_NUMBER, COMP_ITEM_REVISION, COUNTRY_OF_ORIGIN, WORK_CENTER, HISTORY_COMMENT

PfsFindSerialNumber
Req: REQUEST_TYPE, DATABASE, USER_ID, PASSWORD, SERIAL_NUMBER, RETURN_VALUES
Opt: PRODUCTION_ORDER, ITEM_NUMBER, OPERATION_CODE

PfsGenerateSerialNumbers
Req: REQUEST_TYPE, DATABASE, PRODUCTION_ORDER, OPERATION_CODE, WORK_CENTER, STARTING_SERIAL_NUMBER, (ENDING_SERIAL_NUMBER or QUANTITY), BASE
Opt: PREFIX_LENGTH, SUFFIX_LENGTH, INCREMENT, GENERATE_NUMBERS, RETURN_NUMBERS, COMMENT

PfsSetHalt
Req: REQUEST_TYPE, DATABASE, USER_ID, PASSWORD, (PRODUCTION_ORDER or ITEM_NUMBER), SERIAL_NUMBER, HALT_COMMENT
Opt: none

PfsClearHalt
Req: REQUEST_TYPE, DATABASE, USER_ID, PASSWORD, (PRODUCTION_ORDER or ITEM_NUMBER), SERIAL_NUMBER
Opt: none
